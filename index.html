Bug #1 – Mobile Navigation Not Opening

The mobile menu button does nothing when tapped. The user can’t open the navigation.

Cause: The event listener is attached to the wrong element.

Fix: Attach the listener to the correct button and ensure the menu toggles visibility.

Bug #2 – Form Validation Not Working

The form submits even when required fields are empty. The validation logic fails silently.

Cause: Incorrect comparison operator and missing preventDefault().

Fix: Use strict comparison and ensure default form submission is stopped.

Bug #3 – Layout Breaking on Mobile

On small screens, the layout overflows horizontally and forces users to scroll sideways.

Cause: The card container uses fixed widths and no flex-wrap.

Fix: Use flexible sizing, wrapping, and proper responsive breakpoints.
